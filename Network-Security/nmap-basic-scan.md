# Nmap Basic Network Scan

## Objective
Perform basic network reconnaissance using Nmap to identify open ports and services.

## Tool Used
- Nmap

## Commands Used
nmap -sS target_ip
nmap -sV target_ip

## Methodology
1. Performed TCP SYN scan to discover open ports.
2. Identified running services and versions.
3. Analyzed exposed services for potential risks.

## Findings
- Open ports detected on target machine.
- Service versions identified for further enumeration.

## Learning Outcome
Learned how reconnaissance helps attackers and defenders understand network exposure and attack surface.
